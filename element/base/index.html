# base

定義頁面中所有 相對 連結， 應放置在<head>中.

程式範例: Copy

HTML 基礎元素
{% highlight html %} HTML 基礎元素 {% endhighlight %}

href

定義網頁的所有連結。

"{{site.url}}"

可使用 絕對位置 的網址。

target

網頁的連結將顯示的位置。

"_blank"

以 新分頁 的方式開啟。

"_self"

Opens in the current tab.

"_parent"

Opens in the parent browsing context, or _self is there is none.

"_top"

Opens in the top browsing context, or _self is there is none.
